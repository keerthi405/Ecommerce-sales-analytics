# E-commerce Sales Analytics Dashboard

## Project Overview

This project analyzes an E-commerce sales dataset using SQL, Python, and Power BI to generate business insights and build an interactive dashboard.

## Tools Used

- Python
- Pandas
- NumPy
- MySQL
- Power BI
- Excel

## Project Workflow

1. Data Cleaning
2. SQL Analysis
3. Exploratory Data Analysis (EDA)
4. Dashboard Development
5. Business Insights

## Dashboard Features

- KPI Cards
- Monthly Sales Trend
- Category-wise Sales
- Region-wise Sales
- Top Products
- Top Customers
- Interactive Filters

## Skills Demonstrated

- Data Cleaning
- SQL Queries
- Data Visualization
- DAX Measures
- Business Analysis
